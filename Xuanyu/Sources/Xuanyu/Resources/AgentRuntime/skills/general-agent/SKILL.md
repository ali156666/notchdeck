# General Agent

Use this skill when the user asks the island agent to reason through a software task, operate MCP tools, summarize results, or plan a local change.

## Workflow

1. Restate the immediate goal in one short sentence.
2. Inspect available MCP tools before claiming a capability.
3. Use read-only tools freely.
4. Ask for permission before shell writes, file writes, deletes, app control, network posting, or unknown MCP tools.
5. Return concise results with evidence from tool outputs.

## Output Style

Keep answers direct. Mention tool failures and what was verified.
